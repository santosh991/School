EXAM MANAGEMENT:
    Downloading marksheet
  1. Locate the menu named "STAFF" and click on it,a parnel will open.
  2. Locate the staff for which you want to download the marksheet.
  3. See the column named "subjects" and click on it.
  4. Subjects belonging to the selected staff and the corresponding class will be listed.
  5. See the colums named "Scores" and "Download".
  6. See the button named "Excel xlsx" and "Excel xls on the "download" column.
  7. Clicking on either of these buttons will download an excel sheet.
  8. Notice that the excel file is named subjetCode.classCode.examCode.extension
  9. Open the excel file.
  10. The excel containes admission numbers for students in a particular class.
  11. Feed the marks against the admission numbers.
  12. Save the excel file and Close it.
___________________________________________________________________________________________________________________________________
Filling the marks
  1. Once you open the file, see the column "Adm No" with admission numbers for student in the particular class.
  2. See also the column "Scores-30", Note: The value 30 means that the exam is out of 30.
  3. Modify the value to a value of your choice,NB: The exam will be computed out of that value.
___________________________________________________________________________________________________________________________________
Upload exam
  1. Locate the menu named "STAFF" and click on it,a parnel will open.
  2. Locate the staff for which you want to upload exam .
  3. See the column named "subjects" and click on it.
  4. Now see the button "Upload Exam" ,click on it.
  5. A new parnel opens, See the button "Choose file" and click on it.
  6. Select the excel file you have just prepared (or any other file) and click "Upload File".
  7. Note the Message returned. If it's a success message, you can proceed to view the score.
  8. Otherwise correct the error and upload the file again.
___________________________________________________________________________________________________________________________________
View the score
  1. Locate the menu named "STAFF" and click on it,a parnel will open.
  2. Choose a staff of your choice. See the column named "subjects" and click on it.
  3. Subjects belonging to the selected staff and the corresponding class will be listed.
  4. See the button named "view" on the "Scores" column.
  5. Clicking on this button will help you view the just uploaded scores for the selected subject.
___________________________________________________________________________________________________________________________________
Exam results
  1. Locate the menu named "STAFF" and click on it,a parnel will open.
  2. See the sub-menu named "Class Teachers" and click on it.
  3. A new parnel will open with the list of all class teachers with their corresponding classes.
  4. See the following buttons "Results","ClassList","FeeList","Change Class".
    5. Results
  5. Click on this button to download and/or print class performance list and student report card.
  6. For form 3 and 4, see the follwing buttons:
    • Standard ClassList: Class list that comprises paper1,paper2 and paper 3 exams.
    • Non-Standard ClassList: Class list that comprises cat1,cat2 and endTerm exams.
    • Standard ReportCard: Report card that comprises paper1,paper2 and paper 3 exams.
    • Non-Standard ReportCard: Report card that comprises cat1,cat2 and endTerm exams.
  7. Click on the desired button to download the report.
    8. NB:
    The button FeeList will download excel with students fee balance for the particular class.
    The button ClassList will download classList with students basic details.
    The other button Change Class is used to change the class teacher.
